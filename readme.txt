== Description ==

This plugin allows you to use a short code for hidden your some content for the members only to see them.
Just type in your post area like [membersonly]Text to hide[/membersonly]
<a href="http://www.thlive.com" title="thlive" target="_blank">The demo site</a>


== Installation ==

1. Copy the `member-read-only` directory into your `wp-content/plugins` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. That's it! :)
